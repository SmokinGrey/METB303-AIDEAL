# 🛍️ AIDEAL (개발 중)

> **AI가 만드는 이상적인(IDEAL) 거래(DEAL)**

**AIDEAL**은 중소 쇼핑몰을 위한 AI 쇼핑 어시스턴트 SaaS 솔루션입니다. 고객에게는 대화를 통한 개인화된 쇼핑 경험을 제공하고, 쇼핑몰 운영자에게는 자동화된 고객 응대 및 매출 증대 기회를 제공하는 것을 목표로 합니다.

**현재 이 프로젝트는 활발히 개발 중입니다.**

---

## 🎯 프로젝트 목표

- **문제**: 중소 쇼핑몰은 CS 응대, 상품 추천, 구매 전환율 증대에 어려움을 겪고 있으며, 고가의 AI 솔루션 도입이 어렵습니다.
- **해결책**: 합리적인 월 구독료로 AI 쇼핑 어시스턴트를 제공하여, 24시간 고객 응대, 맞춤 상품 추천, 장바구니 연동까지 원스톱으로 지원합니다.

---

## 🌟 주요 기능 (MVP)

AIDEAL의 MVP(최소기능제품)는 다음 핵심 기능들을 포함할 예정입니다.

- **AI 쇼핑 어시스턴트**: 자연어 대화를 통해 고객의 의도를 파악하고, 쇼핑몰 상품 정보와 연동하여 최적의 상품을 추천합니다.
- **원클릭 설치**: 쇼핑몰 운영자가 관리자 페이지에서 스크립트 한 줄을 복사하여 붙여넣는 것만으로 간단하게 설치할 수 있습니다.
- **관리자 대시보드**: AI 챗봇의 활동 통계, 고객 질문 로그, 추천 상품에 대한 반응 등 기본적인 분석 데이터를 제공합니다.
- **쇼핑몰 플랫폼 연동**: MVP 단계에서는 **카페24**와의 연동을 우선 목표로 하며, 상품/재고 정보의 실시간 동기화를 지원합니다.

## 🛠️ 기술 스택

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: Supabase
- **AI Engine**: OpenAI GPT-4o-mini
- **Hosting**: Vercel (Frontend), AWS Lambda (Backend)
