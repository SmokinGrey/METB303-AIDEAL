## 🚀 Quick Start

### For Store Owners (Non-Technical)

1. **Sign up** at [aideal.ai](https://aideal.ai)
2. **Connect your store** (select Cafe24, Godo Mall, etc.)
3. **Customize your AI**:
   - Set brand tone (formal/casual)
   - Choose priority products
   - Configure appearance
4. **Copy & paste** the installation script
5. **Go live** - Your AI assistant is ready!

**Free 7-day trial** - No credit card required

### For Developers

```bash
# Clone the repository
git clone https://github.com/yourusername/aideal.git

# Install dependencies
cd aideal
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

# Run development server
npm run dev

# Build for production
npm run build
```

## 🙋 FAQ

<details>
<summary><strong>How is AIDEAL different from other chatbots?</strong></summary>

Most chatbots just answer FAQs. AIDEAL is a complete shopping assistant that:
- Understands complex product queries
- Makes intelligent recommendations
- Handles real-time inventory
- Completes transactions within the chat
- Learns from each conversation

</details>

<details>
<summary><strong>Will it work with my existing store?</strong></summary>

Yes! If you're using Cafe24, Godo Mall, Makeshop, or Naver Smart Store, integration is automatic. For other platforms, we offer custom integration (contact us).

</details>

<details>
<summary><strong>Do I need coding knowledge?</strong></summary>

Absolutely not! Store owners can install AIDEAL by copying and pasting one line of code. Our dashboard handles all customization through a visual interface.

</details>

<details>
<summary><strong>How accurate are the AI recommendations?</strong></summary>

AIDEAL uses GPT-4o-mini with custom e-commerce training. Accuracy improves over time as it learns from your store's data. Initial accuracy: ~85%, after 1 month: ~92%.

</details>

<details>
<summary><strong>What if AI gives wrong information?</strong></summary>

You can review all conversations in the dashboard and provide corrections. The AI learns from your feedback. You can also set up approval workflows for sensitive information.

</details>

<details>
<summary><strong>Is customer data secure?</strong></summary>

Yes. We're GDPR and PIPA (Korean Personal Information Protection Act) compliant. All data is encrypted, and we never share customer information with third parties.

</details>

<details>
<summary><strong>Can I try before buying?</strong></summary>

Absolutely! Sign up for a free 7-day trial. No credit card required. You get full access to all Pro features during the trial.

</details>
